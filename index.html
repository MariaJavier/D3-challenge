Correlations Discovered Between Poverty, Age, Income and Health Risks

The data set included is based on 2014 ACS 1-year estimates showing rates of income, obesity, poverty, MOE (margine of error), etc. by state.

For this d3 challenge, I have created a scatter plot between Healthcare and Poverty. The code within app.js pulls data from the data.csv.

For extra visualizations, I have added the following:

  1) Tool Tips via mouse hovers over each circle to reveal:
        • state abbreviations
        • exact # (in percent) for those in poverty
        • exact # (in percent) for those lacking healthcare
  2) Animation of the circles from the top left to their appropriate position on the chart
  3) Although no data was included, I demonstrated that this chart can be expanded upon with multiple interactive charts on one html page. As this is just an example, the other X and Y axes labels have been left as inactive.
